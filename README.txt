Sanitary Tubing Welding Estimator v6

How to use:
1. Unzip the folder.
2. Open index.html in a web browser.
3. Use one row for each tubing size used on the job.
4. Each size row now includes:
   - Tube price per foot
   - Elbow, tee, ferrule, hanger pricing
   - Tri-clamp quantity and price
   - Gasket quantity and price
   - Weld count and weld hours
5. Use Extra Parts / Equipment for pumps, valves, instruments, rentals, and misc items.
6. Labor, travel, PM, waste, material markup, labor markup, and final summary totals calculate automatically.

New in v6:
- Tri-clamps and gaskets are priced by tubing size.
- Extra parts/equipment table added for pumps, valves, and misc items.
- Summary now separates tube, fittings, tri-clamps/gaskets, and extra parts.
